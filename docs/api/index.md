# Referencia de API

Autodoc completo de todos los módulos públicos de IOB-Solve v0.2.0.

## Mapa de Módulos

```
iobsolve/
├── core/        Primitivas matemáticas (tipos, Laplacianos, espacios, espectral, partición)
├── continuous/  Motor del dominio continuo (operador de bisagra, localizador del teorema de flujo)
├── discrete/    Motor del dominio discreto (D-IOB, Z-Score, cirugía)
├── plugins/     Aplicaciones de dominio (Lorenz-96, Escudo DDoS, Colapso Modal)
└── io/          Capa de E/S (parsers, exporters, visualizadores)
```

Utilice la navegación de la izquierda para explorar cada módulo.
